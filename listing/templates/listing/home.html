Hello everyone! This is a first production of a textbook classifieds for University of Idaho. Click search to search by subject and course number. Click post a listing to post a listing.

I put this together in about two days. You should email me if:

  • You want to delete a listing. (autodeletion coming soon)
  • You have a bug to report (this will probably happen).
  • You have web development/design experience (bonus points for python/django) and you'd like to help on this small project. Also useful would be sysadmin experience. You don't need to know what you're doing, you just need to have an interest--this is a learning project.
  • You want to request a feature to be added eventually.

There won't be a lot of listings at first so you should probably just make a few with the textbooks you want to get rid of and forget about it for a bit. It's barebones because of how new it is; eventually I'll go in and make it look pretty.

This will only work if you help squash bugs and participate. Thanks a ton!
